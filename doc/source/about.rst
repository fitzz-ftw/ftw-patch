Typography
----------

We believe that high-quality typography is essential for technical 
understanding. This site uses the **TeX Gyre** font collection, which 
provides a modern, extended version of classic typefaces.

.. list-table:: Typography Metadata
   :widths: 30 70
   :header-rows: 1

   * - Role
     - Typeface
   * - **Body Text**
     - TeX Gyre Pagella (based on Palatino)
   * - **Headings**
     - TeX Gyre Heros (based on Helvetica)
   * - **Code**
     - TeX Gyre Cursor (based on Courier)
